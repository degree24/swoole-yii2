# yii2-s
speed up yii2、restful by swoole

# install (php 7.1.16、swoole 1.9.23)
```
composer -vvv require degree757/yii2-s
```

# run
```
cp -R vendor/degree757/yii2-s/demo ./
php demo/bin/http.php
```
